# 🎮 DELTA FORCE: 3D 戰術訓練基地 (3D Tactical Training Outpost)

這是一個基於 **React 19** + **Vite** + **Three.js** (React Three Fiber) 開發的第一人稱 Web 戰術射擊防守與物資搜刮網頁遊戲。

我們已經對專案的說明文件進行了歸類整理，所有開發日誌與專案報告均採用 `.md` 格式分類，方便閱讀與維護：

---

## 📂 專案文檔導覽與分類

### 📖 1. [遊戲介紹 (Game Introduction)](file:///d:/projects/delta%203D/docs/game_intro.md)
*   **內容包含**：遊戲核心特色、電腦端與行動端的操作指南、以及本地開發與生產建置部署的命令說明。

### 🛠️ 2. [運用技術 (Applied Technologies)](file:///d:/projects/delta%203D/docs/applied_tech.md)
*   **內容包含**：專案核心的四大技術板塊（3D 視覺渲染、介面適配、數據同步與儲存、物理與射擊判定）之運作原理與學習報告。

### 🔍 3. [除錯過程 (Debugging Process)](file:///d:/projects/delta%203D/docs/bug_history.md)
*   **內容包含**：本專案歷史 Bug 的詳細診斷分析、解決方案，以及防範舊 Bug 再次出現（Regression）的預防檢修清單（Checklists）。

### 📅 4. [工作日誌 (Work Log)](file:///d:/projects/delta%203D/docs/work_log.md)
*   **內容包含**：專案的迭代日誌，詳述了從「倉庫改版與裝備修復」、「多指觸碰防縮放」、「HUD 行動端版面重定位與轉向靈敏度調整」，到「管理員帳號持久化」的所有工作記錄。

### 🎓 5. [備審資料與學習歷程 (Portfolio & Dossier Guide)](file:///d:/projects/delta%203D/docs/portfolio_learning_dossier.md)
*   **內容包含**：專門為高中升大學申請入學管道設計的專案成果報告模板。詳細描述了「人機協同開發」模式中學生的 PM/架構師/QA 角色定位、四大核心技術（空間扭曲、防卡死 AI、2FA 驗證、行動端適配）的設計思路，以及 17 項防回歸測試清單所體現的工程素養。

## 💡 AI 開發指南說明
> **重要指令**：本專案所有未來的說明文字、進度報告或技術記錄，**均必須以 `.md` 格式寫入對應的分類文檔中**：
> *   新增功能或改動日誌：記錄至 [工作日誌](file:///d:/projects/delta%203D/docs/work_log.md)。
> *   修復 Bug 或新增 Checklist：記錄至 [除錯過程](file:///d:/projects/delta%203D/docs/bug_history.md)。
> *   新增系統、黑市物品或配件：記錄至 [遊戲介紹](file:///d:/projects/delta%203D/docs/game_intro.md)。
> *   引入新技術、庫或框架：記錄至 [運用技術](file:///d:/projects/delta%203D/docs/applied_tech.md)。
