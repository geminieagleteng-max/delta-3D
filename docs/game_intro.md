# 🎮 DELTA FORCE: 3D 戰術訓練基地 - 遊戲介紹

歡迎來到 **Delta Force: 3D Tactical Training Outpost (3D 戰術訓練基地)**！這是一款輕量且流暢的第一人稱 3D 射擊防守與物資搜刮網頁遊戲。

---

## 🌟 遊戲核心特色
1. **多重操控端支援**：支援電腦桌機鍵盤滑鼠操作，同時具備專為行動端（手機、平板）設計的虛擬搖桿與按鈕。
2. **戰術配裝系統 (Loadout)**：在進入戰局前，玩家可於大廳自由裝備主武器、副武器、防彈衣、戰術頭盔、手榴彈與醫療包。
3. **改裝槍械系統 (Gunsmith)**：支援拖曳改裝槍械配件，如紅點瞄準鏡、4倍鏡、消音器、垂直握把、擴容彈匣等。
4. **格狀戰術倉庫 (Grid Stash)**：類似《逃離塔科夫》的格狀拖曳式倉庫系統，整理您的戰利品與裝備，按分類（槍械、裝具、消耗品、戰利品）清晰管理。
5. **黑市商店與特工合約 (Market & Contracts)**：完成局內特工任務賺取 Delta 幣，並在黑市購買高階武器與配件，或出售搜刮的黃金、硬碟等戰利品。
6. **動態波次防守 (Wave Defense)**：多波次逐漸增強的敵軍攻勢。第 3 波攻勢後將指引直升機撤離點（LZ），玩家必須生存並走到撤離區以獲得戰術撤離勝利。
7. **管理員特權控制台**：管理員專屬的 CRT 控制面板與鍵盤作弊代碼，便於在測試與開發時調整波次、滿血或通關。

## 🚇 地鐵長廊（8號出口）玩法與隨機事件
遊戲除了標準的「戰術前哨基地」防守波次撤離模式外，還新增了**「地鐵長廊（地下通道）」**專屬玩法。

1. **8號出口通關規則**：
   - 玩家進入地下通道地圖後，將身處「出口 8」的長廊。
   - 每一區會生成一波敵人。玩家必須將當前長廊的所有敵方 AI 徹底肅清，然後前往長廊盡頭走廊前進下一區。
   - 每前進一區，盡頭的指示牌數字將會遞減（8 ➔ 7 ➔ 6 ... ➔ 1）。
   - 當肅清「出口 1」的敵人後，鐵捲門將會開啟，玩家可前往盡頭走樓梯成功戰術撤離。
2. **每次進圖隨機事件 (Random Events)**：
   每次前進至下一區時，系統將有機會隨機觸發以下特殊環境事件：
   * **正常狀態 (Normal)**：地鐵長廊燈光充足，局勢平穩。
   * **電力中斷 (Blackout)**：地鐵日光燈熄滅，環境一片漆黑。此時玩家槍上會自動開啟「戰術手電筒」提供錐形照明，考驗黑暗中的接敵反應。
   * **化學濃霧 (Dense Fog)**：長廊內瀰漫綠色的化學濃霧，視野能見度大幅降低，敵我雙方更易在近距離遭遇。
   * **紅色警報 (Red Alert)**：長廊警報大作，閃爍著紅色應急燈，並伴隨低沉的警報蜂鳴聲，戰鬥氛圍極度緊張。
   * **空間扭曲 (Space Warp)**：長廊結構發生非線性異常彎曲，筆直的通道扭曲成 S 型正弦波曲線。所有的牆面、天花板、地面日光燈、指示牌及垃圾桶等掩體均會同步位移旋轉，且物理碰撞邊界與 AI 行動路徑皆會對齊彎曲通道。
   * **交戰區域 (Combat Zone)**：地鐵內生成我方 **DELTA 特種部隊友軍**。友軍穿戴青色戰術裝甲與藍色頭盔，發射青色彈道，並在頭頂顯示「DELTA 友軍」健康條。他們會自主尋找並交火敵方 AI，協助玩家肅清通道。友軍不會受到玩家的擊傷判定，亦不計入關卡肅清敵軍的計數。

## 🔧 槍械改裝配件 (Attachments)
玩家可以拖曳配件至槍械插槽中以強化武器性能：

1.  **紅點瞄準鏡 (Sight)**
    *   *效果*：提供發光紅點視野，**開鏡速度提升 15%**。
2.  **ACOG 4倍瞄準鏡 (Sight)**
    *   *效果*：提供中距離望遠十字視野，開鏡速度降低 5%，開鏡 FOV 縮小至 32。
3.  **戰術消音器 (Muzzle)**
    *   *效果*：消除槍口火光並抑制作戰槍響，**垂直後座力減少 10%**。
4.  **槍口補償器 (Muzzle)**
    *   *效果*：大幅控制槍口上跳，**垂直與水平後座力減少 25%**，開鏡速度降低 5%。
5.  **戰術垂直握把 (Grip)**
    *   *效果*：**射擊擴散減少 20%**，**水平後座力減少 15%**。
6.  **人體工學斜角握把 (Grip)**
    *   *效果*：開鏡瞄準動作更為流暢，**開鏡速度大幅提升 30%**。
7.  **雙排擴容彈匣 (Magazine)**
    *   *效果*：增加彈匣容量 (**步槍/衝鋒槍 +10 發，手槍 +5 發**)，但裝彈速度降低 15%。
8.  **快速拔裝彈匣 (Magazine)**
    *   *效果*：加速戰術換彈，**裝彈速度提升 30%**。

## ⌨️ 電腦操作說明
*   **移動**：`W` `A` `S` `D`
*   **視角轉向**：移動滑鼠 (點擊畫面進入滑鼠鎖定)
*   **射擊**：滑鼠左鍵 `L-Click`
*   **瞄準 (ADS)**：滑鼠右鍵 `R-Click`
*   **裝彈**：`R`
*   **下蹲**：`C` (降低視角與隱蔽性，視角高度由 1.6 米降至 0.9 米)
*   **投擲手榴彈**：`G`
*   **使用醫療包**：`H`
*   **互動/搜刮物資**：`F` (靠近加密箱或木箱)
*   **跳躍**：空白鍵 `Space`
*   **武器切換**：`1` (主武器) / `2` (副武器)
*   **管理員作弊快捷鍵** (限管理員帳號)：
    *   `O` 鍵：跳過當前波次，直接進入下一波。
    *   `P` 鍵：滿血恢復。
    *   `K` 鍵：一鍵勝利通關。

## 🛡️ 管理員雙重驗證 (2FA) 登入說明
為了維護管理員特權的安全性，管理員帳號已啟用**雙重安全認證登入 (2FA)**：
1. **常規限制**：無法再使用一般的「使用者帳號 / 密碼」方式登入 `distant star` 管理員帳號。若嘗試登入，系統會引導您切換登入分頁。
2. **登入步驟**：
   * 在主登入選單切換至 **【金鑰登入 KEY】** 分頁。
   * 輸入遊戲金鑰：`DELTA-ADMIN-9999-STAR` (不分大小寫)。
   * 輸入帳號密碼：`f132976655`。
   * 輸入動態驗證碼：輸入下方軍規 **「電子密碼安全器 (SECURE TOKEN)」** 顯示的即時 6 位數動態驗證碼（每 30 秒自動更新一次，備有時差容差，防止臨界點失效）。
   * 點擊「雙重安全驗證 VERIFY & LOGIN」即可驗證成功登入管理員。

## 🛠️ 開發與建置部署
本專案採用 **Vite** 作為前端建置工具，**React 19** 作為開發框架。

*   **本地開發伺服器**：
    ```bash
    npm run dev
    ```
*   **生產環境打包**：
    ```bash
    npm run build
    ```
*   **預覽打包後網頁**：
    ```bash
    npm run preview
    ```
