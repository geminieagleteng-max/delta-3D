# 🎮 DELTA FORCE: 3D 戰術訓練基地 - 遊戲介紹

歡迎來到 **Delta Force: 3D Tactical Training Outpost (3D 戰術訓練基地)**！這是一款輕量且流暢的第一人稱 3D 射擊防守與物資搜刮網頁遊戲。

---

## 🌟 遊戲核心特色
1. **多重操控端支援**：支援電腦桌機鍵盤滑鼠操作，同時具備專為行動端（手機、平板）設計的虛擬搖桿與按鈕。
2. **戰術配裝系統 (Loadout)**：在進入戰局前，玩家可於大廳自由裝備主武器、副武器、防彈衣、戰術頭盔、手榴彈與醫療包。
3. **改裝槍械系統 (Gunsmith)**：支援拖曳改裝槍械配件，如紅點瞄準鏡、4倍鏡、消音器、垂直握把、擴容彈匣等。
4. **格狀戰術倉庫 (Grid Stash)**：類似《逃離塔科夫》的格狀拖曳式倉庫系統，整理您的戰利品與裝備，按分類（槍械、裝具、消耗品、戰利品）清晰管理。
5. **黑市商店與特工合約 (Market & Contracts)**：完成局內特工任務賺取 Delta 幣，並在黑市購買高階武器與配件，或出售搜刮的黃金、硬碟等戰利品。
6. **動態波次防守 (Wave Defense)**：多波次逐漸增強的敵軍攻勢。第 3 波攻勢後將指引直升機撤離點（LZ），玩家必須生存並走到撤離區以獲得戰術撤離勝利。
7. **管理員特權控制台**：管理員專屬的 CRT 控制面板與鍵盤作弊代碼，便於在測試與開發時調整波次、滿血或通關。

## ⌨️ 電腦操作說明
*   **移動**：`W` `A` `S` `D`
*   **視角轉向**：移動滑鼠 (點擊畫面進入滑鼠鎖定)
*   **射擊**：滑鼠左鍵 `L-Click`
*   **瞄準 (ADS)**：滑鼠右鍵 `R-Click`
*   **裝彈**：`R`
*   **下蹲**：`C` (降低視角與隱蔽性)
*   **投擲手榴彈**：`G`
*   **使用醫療包**：`H`
*   **互動/搜刮物資**：`F` (走近保險箱或箱子)
*   **跳躍**：空白鍵 `Space`
*   **武器切換**：`1` (主武器) / `2` (副武器)
*   **管理員作弊快捷鍵**：
    *   `O` 鍵：跳過當前波次，直接進入下一波。
    *   `P` 鍵：滿血恢復。
    *   `K` 鍵：一鍵勝利通關。

## 🛠️ 開發與建置部署
本專案採用 **Vite** 作為前端建置工具，**React 19** 作為開發框架。

*   **本地開發伺服器**：
    ```bash
    npm run dev
    ```
*   **生產環境打包**：
    ```bash
    npm run build
    ```
*   **預覽打包後網頁**：
    ```bash
    npm run preview
    ```
